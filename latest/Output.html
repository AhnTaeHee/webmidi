new Output( midiOutput )

Description

The Output object represents a MIDI output port on the host system. This object is created by the MIDI subsystem and cannot be instantiated directly.

You will find all available Output objects in the WebMidi.outputs array.

Parameters
Name Type Description
midiOutput MIDIOutput

Actual MIDIOutput object as defined by the MIDI subsystem
